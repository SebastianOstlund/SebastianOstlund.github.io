  • My own scripting language

    • My scripting language allows you to add or change code without redeploying or restarting the app. A common disadvantage for scripting languages is that they are often quite slow. This is solved by introducing a 'script to C#' translation unit in my programming language, which lets the code be executed at the same speed as the rest of your program when you rebuild the app. It includes code completion and error handling, and can be added to a Unity project as a console or ingame computer screen.

    Role: Solo Developer
    Tags: Parser, Assembly, Virtual Machine, Byte Code, Stack Machine
